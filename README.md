# Proceso de desarrollo de una aplicación web con ASP.NET

## 1. Creación del proyecto de Visual Studio

1.1 Abra Visual Studio y seleccione **Archivo -> Nuevo Projcto** en el menú superior. ![menu_new_project.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/menu_new_project.png)

1.2 Seleccione ** plantilla de sitio web ASP.NET** en la sección **Carácterísticas del proyecto**. ![plantilla_site_web_vb.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/plantilla_site_web_vb.png)

1.3 En el cuadro de diálogo **Nueva Plantilla** seleccione una plantilla y haga clic en **Aceptar**. ![nuevo_plantilla.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/nuevo_plantilla.png)

1.4 Configure las opciones de la plantilla deseadas y haga clic en **Aceptar**. ![configura_proyectos.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/configura_proyectos.png)

1.5 Configure el nombre, la ubicación y el tipo de proyecto desired (Web Forms, Web API, etc.). ![configure_project.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/configure_project.png)

1.6 Una vez finalizado el proceso de creación del proyecto, verifique la estructura del proyecto en la solución explorer. ![solution_explorer.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/solution_explorer.png)

## 2. Creación de una aplicación web con ASP.NET

2.1 Abra el proyecto creado en el paso anterior. ![open_project.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/open_project.png)

2.2 Haga clic en **Agregar** y seleccione **Elemento de sitio web**. ![add_web_site.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/add_web_site.png)

2.3 Configure las opciones deseadas, como el nombre de la aplicación y el tipo de proyecto (Web Forms en este caso). ![configure_project.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/configure_project.png)

2.4 Haga clic en **Aceptar**. ![accept.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/accept.png)

2.5 Verifique que se haya creado una aplicación web con plantilla de sitio web ASP.NET. ![web_site_project.png](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/web_site_project.png)

## 3. Creación y edición de una página web

3.1 Abra el archivo Login.aspx en el editor de código fuente del proyecto. ![open_file_login.aspx](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/open_file_login.aspx)

3.2 Verifique que la página web tenga un formulario de entrada y una vista previa asociada a la página. ![form_login](https://github.com/andresmoreira022/Tecnicas-Web-I/blob/master/images/form_login)

3.3 Edite el contenido del archivo Login.aspx para ajustarlo según las necesidades de la aplicación.
